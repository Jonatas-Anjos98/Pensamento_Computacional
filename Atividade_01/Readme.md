# 📚 Sistema de Administração de Atividades Escolares

Este projeto foi desenvolvido como atividade da disciplina **Pensamento Computacional** do curso de **Análise e Desenvolvimento de Sistemas** do Centro Universitário UDF.

O objetivo do projeto é aplicar os conceitos de **pensamento computacional** para resolver um problema real do dia a dia: **organizar e administrar atividades escolares**.

---

# 🎯 Problema

Muitos estudantes enfrentam dificuldades para organizar suas tarefas acadêmicas, como:

- trabalhos
- provas
- exercícios
- prazos de entrega
- materiais de estudo

A falta de organização pode causar **atrasos, acúmulo de tarefas e estresse**.

Este projeto propõe uma solução simples para **gerenciar e acompanhar atividades escolares de forma organizada**.

---

# 🧠 Aplicação do Pensamento Computacional

## 1️⃣ Decomposição

O problema foi dividido em pequenas partes para facilitar a solução:

- cadastrar atividades
- definir prazos de entrega
- organizar atividades por disciplina
- acompanhar tarefas concluídas
- visualizar tarefas pendentes
- atualizar ou remover atividades

---

## 2️⃣ Abstração

Alguns detalhes foram simplificados para focar na essência do problema.

A essência do sistema é:

> **Registrar, organizar e acompanhar atividades escolares de forma simples e eficiente.**

Assim, o sistema se concentra apenas nas funções principais necessárias para gerenciar tarefas.

---

# ⚙️ Funcionalidades

O sistema pode incluir funcionalidades como:

- 📌 Cadastro de atividades
- 📅 Definição de prazos
- 📚 Organização por disciplina
- ✅ Marcação de tarefas concluídas
- ✏️ Edição de atividades
- 🗑 Remoção de atividades

---

# 🚀 Objetivo do Projeto

Este projeto tem como objetivo:

- aplicar conceitos de **pensamento computacional**
- desenvolver **habilidades de organização e resolução de problemas**
- praticar **modelagem de soluções para problemas reais**
