# Pensamento Computacional

Este repositório foi criado para armazenar e organizar as atividades desenvolvidas na disciplina **Pensamento Computacional**, do curso de **Análise e Desenvolvimento de Sistemas** do Centro Universitário UDF.

---

## 📚 Sobre a disciplina

A disciplina de **Pensamento Computacional** tem como objetivo desenvolver habilidades de resolução de problemas utilizando conceitos fundamentais da ciência da computação, como:

- Abstração
- Decomposição de problemas
- Algoritmos
- Lógica computacional
- Modelagem de soluções

Esses conceitos ajudam a estruturar o pensamento de forma lógica e eficiente, permitindo a criação de soluções para problemas complexos.

---

## 🎯 Objetivo do repositório

O objetivo deste repositório é:

- Armazenar as **atividades propostas na disciplina**
- Organizar os **exercícios e projetos desenvolvidos**
- Registrar o **processo de aprendizagem ao longo do semestre**
- Manter um histórico das práticas realizadas durante o curso

---

## 📂 Estrutura do repositório

Este repositório pode conter:

- Exercícios práticos
- Projetos da disciplina
- Materiais de apoio
- Anotações e estudos relacionados ao conteúdo

A organização das pastas pode seguir a estrutura das atividades ou dos módulos da disciplina.

---

## 🛠 Tecnologias utilizadas

Dependendo das atividades propostas na disciplina, poderão ser utilizadas diferentes ferramentas e linguagens, como:

- Algoritmos
- Pseudocódigo
- Linguagens de programação
- Ferramentas de desenvolvimento
